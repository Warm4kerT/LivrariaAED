O nosso projeto foi desenvolvido para ser compilado através de um Make\+File, portanto, inicialmente vamos escrever na nossa linha de comandos \char`\"{}make\char`\"{}, para prepararmos todos os ficheiros executáveis.

Seguidamente, vamos executar o nosso ficheiro principal, ou seja, a nossa \char`\"{}livraria.\+h\char`\"{} utilizando o seguinte comando\+: \char`\"{}./livraria\char`\"{}

Resumindo, Comandos Iniciais\+:


\begin{DoxyEnumerate}
\item \char`\"{}make\char`\"{}
\item \char`\"{}./livraria\char`\"{}
\end{DoxyEnumerate}

Após ambos os passos, o nosso programa dará início, e aparecerá ao utlizador um pequeno Menu Inicial\+:

 Para utilizar este e os demais Menus presentes no projeto, o utilizador apenas tem de digitar na linha de comandos, o número referente à operação que deseja efetuar, seja entrar num Sub-\/\+Menu, Criar um cliente, ou utlizar uma das operações presentes.

Seguidamente, fica um exemplo de como criar um cliente e guardar na nossa Base de Dados. (Exemplifica apenas como criar um \mbox{\hyperlink{structCliente}{Cliente}}, as outras operações do Projeto são utilizadas praticamente da mesma forma, portanto iremos apenas demonstrar esta).

Antes de entrarmos no nosso Sub\+Menu Clientes, temos de entrar no Sub\+Menu Ficheiros para inicializarmos os nossos ficheiros que servem como Base de Dados\+:
\begin{DoxyEnumerate}
\item Entramos no Sub\+Menu escrevendo na linha de comandos o número \char`\"{}1\char`\"{}.
\item Após isso selecionamos a opção Abrir e selecionamos Abrir Clientes\+:
\begin{DoxyEnumerate}
\item Ao selecionar, escrevemos o nome do ficheiro, que neste caso é \char`\"{}clientes.\+txt\char`\"{}.
\item Após estar devidamente aberto podemos sair do Sub\+Menu Ficheiros e seguir para o próximo Passo.
\end{DoxyEnumerate}
\end{DoxyEnumerate}

 Após estar adicionado o \mbox{\hyperlink{structCliente}{Cliente}} á lista de Clientes aberta no início através do ficheiro, temos de guardar esta nova lista atualizada, para também atualizar o nosso ficheiro de Base de Dados de Clientes.
